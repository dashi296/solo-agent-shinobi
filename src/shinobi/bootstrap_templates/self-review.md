# Self Review Checklist

## Goal
- [ ] 目的を満たしている
- [ ] 完了条件を yes/no で満たしている

## Scope
- [ ] スコープ外を混ぜていない
- [ ] 対象外の大きな変更は follow-up に分離した

## Safety
- [ ] failure path の cleanup を確認した
- [ ] state / lock / label の整合性を確認した

## Quality
- [ ] 正常系を検証した
- [ ] 失敗系を検証した
- [ ] lint / typecheck / test の実行結果を確認した

## Docs
- [ ] README / docs との不整合がない
- [ ] 必要なドキュメント更新を含めた
